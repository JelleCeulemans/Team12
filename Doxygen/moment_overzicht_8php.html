fullCalendar is een plugin voor kalender weer te geven

Parameters
themeSystemHiermee kan je verschillende theme's weergeven
defaultViewHiermee kan je de view veranderen, deze is nu ingesteld met view week
nowIndicatorHiermee komt er indiciator die de dag van vandaag weergeeft
minTimeHiermee kan je instellen vanaf welk uur de view begint
maxTimeHiermee kan je instellen vanaf welk uur de view eindigt
headerHiermee kan je header maken die boven de kalender verschijnt
customButtonsHiermee kan je custumbuttons toevoegen, ik heb de button "toevoegen" toegevoegd
editableDoor deze waarde op true te zetten kan je de kalendar bewerken
resizableDoor deze waarde op true te zetten kan je de duur van de event aanpassen
eventOverlapDoor deze waarde op false te zetten kunnen events niet met elkaar overlappen
eventClickWanneer je op de event klikt start deze een functie
eventDropWanneer er met event word gesleept zal deze functie gestart worden wanneer het event gedropt word
eventResizeWanneer de duur van het event word verandert zal deze een functie starten
eventSourcesHiermee kan je de event inladen die vanuit de controller uit de database worden gehaald
